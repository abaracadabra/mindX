# Five concentric ICP rings

Each ring sees a different draft. Same campaign, five renderings —
experimentation_a maintains them as variants tagged with `audienceClusterHash`.

## Ring A — senior crypto-native developers

The ones who fork the SDK, build agents, list them on AgenticPlace.
Convert through GitHub READMEs, Farcaster /aiagents, hackathons, dense
long-form. Voice: technical, direct, citation-dense. No hand-holding.
Surfaces: GitHub, Mirror, ETHGlobal blog, Decentralised.co.

## Ring B — AI researchers and dev-tool power users

Coming from Cursor / Claude Code / Aider / OpenDevin. They want a cognitive
layer they can own. Convert through Latent Space, Replit-style demo videos
pinned on X, the Anthropic-MCP server registry. Voice: technical, empirical.
Show benchmarks, show traces, show diffs.

## Ring C — traders and onchain power users

Hyperliquid / Pump.fun / Base App audience. Convert through points programs,
x402-paid agent services, Frame-native demos. Voice: terse, value-prop-first.
Surfaces: Farcaster /degen, X, Telegram trader channels — but never paid
shilling.

## Ring D — mainstream crypto users

Coinbase / Phantom / Pera Wallet audience. Convert through ENS subname
distribution (\*.bankon.eth, \*.pythai.eth) and one-click agent rental in
BANKON SATPAY-bridged stablecoins. Voice: friendly, concrete, low-jargon.
Surfaces: in-wallet experiences, Frame stories, mainstream crypto media.

## Ring E — philosophical and longevity communities

Stoic Twitter, Bryan Johnson / Don't Die scene, Ribbonfarm / SSC / Lighthaven
readership. Convert ONLY through founder-led writing, eventually IRL events.
**I do not draft for this ring**; it maps to the `code_as_dojo` pillar which
is reserved for the human Convener.
