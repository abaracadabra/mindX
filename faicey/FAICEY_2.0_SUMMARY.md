## 🚀 Running the System

### **Full Installation**
```bash
# Setup voice system (requires sudo)
./scripts/setup-voice-system.sh

# Install dependencies
npm install

# Run enhanced Jaimla demo
npm run enhanced-jaimla
```

### **Lightweight Demo (No Dependencies)**
```bash
# Run compatibility demo
node examples/lightweight-demo.js

# Open browser to http://localhost:8080
```

### **Available Scripts**
- `npm run jaimla` - Basic Jaimla agent demo
- `npm run enhanced-jaimla` - Full-featured enhanced demo
- `npm run voice-test` - Voice creation system test
- `npm run oscilloscope` - D3.js oscilloscope standalone demo
