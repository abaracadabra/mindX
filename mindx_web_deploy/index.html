Help

        
Deployment and configuration.

        

          
Apache

          
Upload this folder to your document root (e.g. public_html on Hostinger). Ensure AllowOverride All if using .htaccess.

        

        

          
Hostinger (shared)

          
Static files only. Set API_BASE in js/config.js to your mindX backend URL (e.g. on a VPS).

        

        

          
VPS

          
Run the mindX backend (uvicorn) and optionally serve this UI via Apache/Nginx in front. You can run custom services and execute commands on the VPS.

        

        

          
Tauri + Web3D

          
Tauri builds a desktop app. Use this UI (or a Tauri webview) to deliver Web3D; the Tauri app runs on the user’s machine, not on Hostinger. See DEPLOY.md.
