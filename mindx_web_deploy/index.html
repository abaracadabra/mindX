Help

        
Deployment and configuration for Hostinger and VPS.

        

          
agenticplace.pythai.net (UI)

          
This is the public UI. Upload this folder to Hostinger (or serve from VPS). Set API_BASE in js/config.js to https://mindx.pythai.net (or your backend URL). On agenticplace.pythai.net it defaults automatically.

        

        

          
mindx.pythai.net (Backend)

          
Run the mindX backend on a VPS: uvicorn mindx_backend_service.main_service:app --host 0.0.0.0 --port 8000. Put Nginx/Apache in front with SSL. See DEPLOY.md.

        

        

          
Apache / Hostinger

          
Upload this folder to document root. Ensure AllowOverride All if using .htaccess. Enable SSL for agenticplace.pythai.net.

        

        

          
Tauri + Web3D

          
Tauri builds a desktop app. Use this UI (or a Tauri webview) to deliver Web3D; the app runs on the user’s machine. See DEPLOY.md.
