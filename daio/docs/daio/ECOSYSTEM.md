# DAIO ecosystem and references

External projects and references used by or related to DAIO.

## GitHub

| Link | Description |
|------|-------------|
| [interplanetaryfilesystem](https://github.com/interplanetaryfilesystem) | IPFS core and ecosystem; reference for decentralized storage and content-addressed data. |
| [ipNFTfs](https://github.com/ipNFTfs) | Interplanetary NFT file system and IPFS utilities; reference for NFT-on-IPFS and DAIO identity/NFT workflows. |
| [mlodular](https://github.com/mlodular) | Modular ML / tooling reference. |
| [faicey](https://github.com/faicey) | Reference project. |
| [jaimla](https://github.com/jaimla) | Reference project. |
| [Professor-Codephreak](https://github.com/Professor-Codephreak) | Reference; see also pythai.net sites below. |
| [w3DAIO](https://github.com/w3DAIO) | First public showing of the DAIO code; Decentralized Autonomous Intelligent Organization (scaffold-eth-2, EVM dev stack). |
| [DAONOW](https://github.com/DAONOW) | Reference for DAO tooling and governance patterns. |
| [dairef](https://github.com/dairef) | Programmable DAI reference (DSS, dss-flash, OpenZeppelin, Synthetix); used for DAI/stablecoin utilities and stable cloning/creation. |
| [MakerDAO/dss](https://github.com/makerdao/dss) | Dai Stablecoin System; core DAI mechanics and reference for stablecoin patterns. |
